# 🚀 GSAP First Project

Este projeto é uma aplicação web desenvolvida para praticar animações utilizando o GSAP (GreenSock Animation Platform), uma poderosa biblioteca JavaScript para criação de animações modernas e performáticas.

## 📌 Sobre o Projeto

O objetivo deste projeto é explorar conceitos de animação no front-end, criando interações suaves e visuais atrativos para o usuário.

Foram utilizadas animações com foco em:
- Transições de elementos
- Animações baseadas em scroll
- Sequenciamento com timeline
- Efeitos visuais modernos

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- GSAP
